Nucleus CSS

About

Nucleus is a minimal CSS boilerplate, with icons included. Nucleus is stand-alone; no dependencies whatsoever. It has support for responsive design. Your project will look great in no time.

Nucleus is compatible with all modern browsers without the use of JavaScript.

Note: Plenty of these stylings are attached directly to the DOM-element. Just because it's nice.

Installation

Download the latest version of Nucleus. Unzip and place the file in your css folder.
Pop this link to your head: 


<link rel="stylesheet" type="text/css" href="css/nucleus.css">

That's it. One file, 24 kB. Done.

Responsive grid

Breakpoints are mobile (0–767px) and tablet (768–1024px).

Simply prefix the above classes with mobile- and tablet- for any element.

Example: you have a photo which should be:

  • 25% width in desktop view
  • 50% width on a tablet
  • hidden on a mobile

Markup:


<img src="photo.png" class="quarter tablet-half mobile-hide">

Icons

The 46 most common web icons are included. To avoid vector bloat and to reduce bytesize, each icon is rebuilt and optimized, point for point, then optimized again with svgo. They are embedded as inline svg – no need to link or load separate icon libraries.

Markup: 


<div class="icon bar"></div>
<div class="icon facebook"></div>
etc.
